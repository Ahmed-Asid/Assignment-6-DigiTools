DigiTools Platform

In this project we build an web that let's you buy digital tools online. The  interface features are very user friendly. 

We made this React based webapp in vite with Html, Css and Javascript.Tailwind css and Daisyui were used for style, Json for data, Some react libraries such as lucide icons, react-icons, react-toastify were also used.

Features:
>Dynamic Cart & Real-time State Management
>Interactive product catalog
>Toast notification system
